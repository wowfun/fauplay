# Fauplay 快捷键

本文档列出当前版本已支持的快捷键与生效范围。

快捷键配置统一维护在 [`src/config/shortcuts.ts`](../src/config/shortcuts.ts)。如需改键位，请优先修改该文件，而不是在组件内写死按键判断。

## 应用快捷键（App Shortcuts）

| 快捷键 | 作用 | 说明 |
| --- | --- | --- |
| `Ctrl/Cmd + O` | 选择文件夹 | 打开系统文件夹选择器 |
| `Backspace` | 返回上一级目录 | 仅在未聚焦输入框时生效 |

## 预览快捷键（Preview Shortcuts）

以下快捷键在预览打开且未聚焦输入框时生效，侧栏预览与全屏预览共用同一套规则。
其中 `P/T/[ / ]` 仅在当前预览文件为媒体（图片/视频）时生效，`Space/J/L/R` 仅在当前预览文件为视频时生效。

| 快捷键 | 作用 | 说明 |
| --- | --- | --- |
| `Esc` | 关闭预览 | 优先关闭全屏预览，其次关闭右侧预览面板 |
| `P` | 切换自动播放 | 仅媒体预览可用，统一作用于侧栏与全屏 |
| `T` | 切换遍历模式 | 仅媒体预览可用，在顺序/随机之间切换 |
| `Space` | 切换视频播放/暂停 | 仅视频预览可用，优先控制全屏视频，否则控制侧栏视频 |
| `J` | 视频快退 | 仅视频预览可用，按“快进步长”配置回退，支持长按连续触发 |
| `L` | 视频快进 | 仅视频预览可用，按“快进步长”配置前进，支持长按连续触发 |
| `R` | 循环切换视频倍速 | 仅视频预览可用，按 `1x -> 3x -> 5x -> 0.5x -> 1x` 循环，长按不连续切换 |
| `Delete` | 软删除当前预览文件 | 触发 `fs.softDelete` 提交执行（移动到 `.trash`） |
| `0-9` | 快速标注当前文件 | 仅在 `meta.annotation` 配置了激活 enum 字段时生效，按字段值定义顺序映射 `0..9` |
| `#` | 打开逻辑标签绑定面板 | 仅文件预览且标签管理能力可用时生效，语义等同点击预览头部 `+` |
| `[` | 上一个 | 仅媒体预览可用，按当前遍历模式切换 |
| `]` | 下一个 | 仅媒体预览可用，按当前遍历模式切换 |

说明：
- 视频预览头部支持“快进步长”与“倍速”下拉（步长：`3/5/10s`，倍速：`0.5/1/3/5x`），并与快捷键联动。
- 快进步长与倍速配置会写入本地存储，刷新后保持上次设置；侧栏与全屏共享同一配置。

## 预览遍历模式（顺序/随机）

- 可在右侧预览窗格或全屏预览头部点击“顺序/随机”按钮切换遍历模式。
- 非媒体文件（如文本/压缩包）不会展示播放控件，也不会响应遍历快捷键。
- 遍历模式会同时影响：
  - 预览内“上一个/下一个”按钮
  - 自动播放触发的下一项切换
  - 非自动播放状态下的手动上一张/下一张

## 网格导航（文件列表区域）

| 快捷键 | 作用 |
| --- | --- |
| `ArrowLeft` / `ArrowRight` | 在网格中左右移动选中项 |
| `ArrowUp` / `ArrowDown` | 在网格中上下移动选中项 |
| `A` / `D` | 在网格中左右移动选中项 |
| `W` / `S` | 在网格中上下移动选中项 |
| `PageUp` / `PageDown` | 按页向上/向下跳转选中项 |
| `Ctrl/Cmd + A` | 全选当前可见网格项（文件+目录） |
| `Esc` | 清空当前勾选集合（仅在无打开预览时） |
| `Enter` | 打开当前选中项：目录进入子目录，文件打开全屏预览 |

说明：
- 键盘移动选中项时，若选中的是文件，会同步更新右侧预览面板。
- 当光标在输入框（搜索框等）中时，网格导航快捷键不会触发。
- 当预览已打开时，`Esc` 优先关闭预览，不会清空网格勾选集合。

## 全屏预览说明

- 全屏预览使用与右侧预览面板相同的预览快捷键契约。
