# Fauplay 快捷键

本文档列出当前版本已支持的快捷键与生效范围。

## 全局快捷键

| 快捷键 | 作用 | 说明 |
| --- | --- | --- |
| `Ctrl/Cmd + O` | 选择文件夹 | 打开系统文件夹选择器 |
| `Backspace` | 返回上一级目录 | 仅在未聚焦输入框时生效 |
| `Esc` | 关闭预览 | 优先关闭全屏预览弹窗，其次关闭右侧预览面板 |

## 网格导航（文件列表区域）

| 快捷键 | 作用 |
| --- | --- |
| `ArrowLeft` / `ArrowRight` | 在网格中左右移动选中项 |
| `ArrowUp` / `ArrowDown` | 在网格中上下移动选中项 |
| `PageUp` / `PageDown` | 按页向上/向下跳转选中项 |
| `Enter` | 打开当前选中项：目录进入子目录，文件打开全屏预览 |

说明：
- 键盘移动选中项时，若选中的是文件，会同步更新右侧预览面板。
- 当光标在输入框（搜索框等）中时，网格导航快捷键不会触发。

## 全屏预览弹窗

| 快捷键 | 作用 |
| --- | --- |
| `Esc` | 关闭全屏预览弹窗 |
| `Space` | 视频预览时切换播放状态 |
