# 118 Toolbar Help & Shortcut Viewer 顶部工具栏帮助与快捷键查看规范

## 1. 目的 (Purpose)

定义 Fauplay 顶部工具栏帮助入口与“当前快捷键”查看能力契约，确保：

1. 顶部工具栏提供统一的帮助入口，用于查看当前运行时生效的快捷键。
2. 帮助面板展示的是运行时合并结果，而不是仅展示默认配置文件。
3. 快捷键列表按 `App / Grid / Preview / Tag` 四组稳定展示，并带有用户可见的状态指示。
4. 动态逻辑标签快捷键单独归入 `Tag` 组，不并入 `Preview` 组。

## 2. 范围与非目标 (In Scope / Out of Scope)

范围内：

1. 顶部工具栏 `帮助` 按钮与锚定式面板。
2. “当前快捷键”全量列表展示。
3. 快捷键当前实际绑定值、分组和运行时状态说明。
4. 动态逻辑标签快捷键的独立 `Tag` 分组展示。

范围外：

1. 快捷键编辑、冲突修复、搜索、过滤与自定义排序。
2. 独立帮助页、全局模态或命令面板。
3. 事件级实时键盘监听提示。

## 3. 用户可见行为契约 (User-visible Contract)

1. 顶部工具栏必须新增 `帮助` 按钮。
2. 点击 `帮助` 按钮后，系统必须在按钮下方打开一个锚定式帮助面板。
3. 再次点击 `帮助`、点击面板外部区域或按 `Esc` 时，帮助面板必须关闭。
4. 帮助面板与顶部工具栏其他浮层必须互斥；打开帮助面板时，收藏、历史、目录段下拉与顶部标签过滤浮层必须关闭。
5. 帮助面板首批仅提供一个只读视图：`当前快捷键`。
6. 快捷键列表必须按 `App / Grid / Preview / Tag` 四组固定顺序展示。
7. 每一项至少展示：
   - 动作名称
   - 当前实际绑定键
   - 状态徽标
   - 简短状态说明
8. 内建快捷键即使当前未配置绑定，也必须继续展示，并标记为 `未配置`。
9. 动态逻辑标签快捷键若当前合并结果中存在合法 `tag:` action，则必须展示在 `Tag` 组中，并按 `key/value` 稳定排序。
10. 当动态标签快捷键与预览内建快捷键使用同一绑定时，帮助面板必须保留两条记录，并在该标签项上标明“激活时优先”。

## 4. 状态语义契约 (Status Semantics)

1. 帮助面板中的状态是面向用户的稳定运行时状态，不依赖单次 `keydown` 事件。
2. 面板状态至少包含：
   - `可用`
   - `当前不可用`
   - `未配置`
3. 下列瞬时守卫不得直接作为帮助面板状态维度：
   - 输入框聚焦
   - `event.repeat`
   - `defaultPrevented`
   - 帮助面板自身焦点
4. 内建快捷键状态必须基于当前稳定上下文计算，包括：
   - 当前路径是否可向上导航
   - 当前可见列表是否存在项目
   - 当前是否存在勾选集合
   - 当前是否打开预览
   - 当前预览是否为媒体
   - 当前预览是否为视频
   - 当前预览是否具备软删除能力
   - 当前预览是否具备标注能力
5. `Tag` 组快捷键状态只取决于两项：
   - 标注能力是否可用
   - 目标逻辑标签当前是否存在于 tag options
6. 因此，`Tag` 组快捷键不得因“当前文件已绑定该标签”而改变帮助面板状态；运行时触发仍可保持幂等 no-op。
7. 当标签候选尚未完成加载时，`Tag` 组可暂时显示 `当前不可用`，直到标签存在性可判定。

## 5. 数据与实现契约 (Data / Implementation Contract)

1. 前端必须提供一个统一的快捷键帮助视图模型，将运行时快捷键快照转换为 UI 可展示条目。
2. 视图模型至少包含：
   - `id`
   - `group`
   - `label`
   - `bindings`
   - `statusKind`
   - `statusText`
   - `order`
3. 视图模型的 built-in metadata 必须集中维护，至少定义动作标题、所属组和排序。
4. 帮助面板消费的快捷键数据必须来自运行时合并结果：
   - 内建快捷键来自 unified shortcut store
   - 动态逻辑标签快捷键来自已合并的 `tag:` 配置结果与 tag options 快照
5. 本专题不得改变 `116` 已定义的快捷键配置契约、解析逻辑与触发优先级。

## 6. 功能需求 (FR)

1. `FR-THS-01` 顶部工具栏必须提供 `帮助` 按钮。
2. `FR-THS-02` `帮助` 按钮必须打开锚定式帮助面板，而不是新模态体系。
3. `FR-THS-03` 帮助面板必须展示当前运行时生效的快捷键合并结果。
4. `FR-THS-04` 帮助面板必须按 `App / Grid / Preview / Tag` 四组固定顺序展示快捷键。
5. `FR-THS-05` 每条快捷键记录必须包含动作名称、实际绑定、状态徽标与简短状态说明。
6. `FR-THS-06` 内建快捷键在绑定被配置为 `"none"` 时仍必须展示，并标记为 `未配置`。
7. `FR-THS-07` `Tag` 组快捷键状态只取决于“标注能力是否可用”和“目标逻辑标签是否存在于 tag options”。
8. `FR-THS-08` 帮助面板不得把输入焦点、按键重复与事件抢占显示为快捷键当前不可用。
9. `FR-THS-09` 当动态标签快捷键与预览内建快捷键同键冲突时，帮助面板必须保留两条记录，并在标签项提示“激活时优先”。

## 7. 验收标准 (AC)

1. `AC-THS-01` 顶部工具栏可见 `帮助` 按钮；点击后面板在按钮下方打开，再次点击、点击外部或按 `Esc` 会关闭。
2. `AC-THS-02` 打开帮助面板时，收藏、历史、目录段下拉与顶部标签过滤浮层会关闭。
3. `AC-THS-03` 当 `~/.fauplay/global/shortcuts.json` 或 `<root>/.fauplay/shortcuts.json` 覆盖某个快捷键后，帮助面板显示的是覆盖后的绑定结果。
4. `AC-THS-04` 当某个内建 action 被配置为 `"none"` 时，该动作仍在帮助面板中显示，且状态为 `未配置`。
5. `AC-THS-05` 在根目录时，`返回上一级目录` 显示 `当前不可用`；进入子目录后切换为 `可用`。
6. `AC-THS-06` 当当前无可见网格项时，网格导航与打开类快捷键显示 `当前不可用`；出现项目后切换为 `可用`。
7. `AC-THS-07` 当当前预览是图片时，媒体预览快捷键显示 `可用`、视频专属快捷键显示 `当前不可用`；切到视频后视频专属快捷键切换为 `可用`。
8. `AC-THS-08` 动态逻辑标签快捷键在 `Tag` 组展示；当标注能力不可用时显示 `当前不可用`，当目标逻辑标签不存在时显示 `当前不可用`，两项都满足时显示 `可用`。
9. `AC-THS-09` 当前文件即使已绑定该逻辑标签，帮助面板中的 `Tag` 组状态也不变化；再次触发快捷键仍为幂等 no-op。
10. `AC-THS-10` 当搜索框或其他输入控件聚焦时，帮助面板中快捷键状态不因输入焦点整体退化为 `当前不可用`。

## 8. 默认值与一致性约束 (Defaults & Consistency)

1. 本专题首批默认只读，不提供快捷键编辑能力。
2. `帮助` 面板文案可以本地化为中文，但分组语义固定对应 `App / Grid / Preview / Tag`。
3. `docs/shortcuts.md` 继续记录默认快捷键与配置语法；帮助面板显示的是当前运行时合并结果与状态。
